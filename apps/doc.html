Methods

Search for Exercises

Endpoint: http://localhost:5000/search

Request type: GET

Query parameters:

  • equipment (optional) – e.g., none, barbell
  • muscle (optional) – e.g., legs, chest
  • type (optional) – e.g., cardio, strength

Example request:

GET /search?equipment=none&muscle=legs&type=cardio

Response: JSON list of exercise names that match the filters, e.g.:

{
  "exercises": [
    "burpees",
    "jogging",
    "jumping jacks"
  ]
}
